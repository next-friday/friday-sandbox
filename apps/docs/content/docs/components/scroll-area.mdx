## Purpose

`ScrollArea` is a scrollable container with custom scrollbars that match your
theme. Content goes inside `ScrollArea.Viewport`; add a `ScrollArea.Scrollbar`
for each direction you want to scroll.

## When to use

- A fixed-height or fixed-width container whose content overflows and needs a
  styled scrollbar that matches the design system.
- You want scrollbar visibility to respond to hover or to always be shown.

## When not to use

- Native browser scrollbars are acceptable â€” the extra markup is not worth it.
- The entire page scroll â€” style the document scroll track in CSS instead.

## Sizes

The `size` prop controls the scrollbar thickness. Choose a thinner track for
compact interfaces and a thicker one for touch-friendly targets.

## Variants

The `variant` prop controls when the scrollbars are visible: `hover` shows them
while the pointer is over the area, `always` keeps them visible, and `hidden`
removes them while the area still scrolls.

## Props

`ScrollArea.Root` also accepts all props from the underlying Radix UI
`ScrollArea.Root`, such as `scrollHideDelay`, `dir`, `asChild`, and `nonce`.

| Prop        | Default   | Type                                                                  |
| ----------- | --------- | ---------------------------------------------------------------------- |
| `size`      | `"md"`    | `"xs" \| "sm" \| "md" \| "lg"`<br />Sets the scrollbar thickness.       |
| `variant`   | `"hover"` | `"hover" \| "always" \| "hidden"`<br />Controls when the scrollbars are visible. |
| `className` | â€”        | `string`<br />Additional CSS classes to apply to the scroll area root. |

## Styling

`ScrollArea` is styled through CSS classes, so the look applies to any element that carries them.

### Modifier classes

| Class                            | Type    | Description                                                          |
| --------------------------------- | ------- | --------------------------------------------------------------------- |
| `fri-scroll-area`                | Part    | Root: medium size default.                                            |
| `fri-scroll-area-viewport`       | Part    | Scrollable viewport.                                                   |
| `fri-scroll-area-scrollbar`      | Part    | Scrollbar track.                                                       |
| `fri-scroll-area-thumb`          | Part    | Draggable thumb inside the track.                                     |
| `fri-scroll-area-corner`         | Part    | Corner where vertical and horizontal scrollbars meet.                 |
| `fri-scroll-area-xs`             | Size    | Extra small size.                                                     |
| `fri-scroll-area-sm`             | Size    | Small size.                                                            |
| `fri-scroll-area-md`             | Size    | Medium size (default).                                                |
| `fri-scroll-area-lg`             | Size    | Large size.                                                            |
| `fri-scroll-area-hide-scrollbar` | Variant | Variant `"hidden"`: hides the scrollbar, keeps the area scrollable.    |

## Accessibility

- The scroll area is reachable by keyboard; content inside `ScrollArea.Viewport`
  can be scrolled with <kbd>Tab</kbd>, arrow keys, <kbd>Page Up</kbd> / <kbd>Page Down</kbd>, and <kbd>Home</kbd> / <kbd>End</kbd>.
- The `"hidden"` variant hides the scrollbar visually but does not disable
  scrolling via wheel, touch, or keyboard.
- When the scrollable region contains interactive elements, ensure they are all
  reachable by keyboard without relying on mouse scroll.
