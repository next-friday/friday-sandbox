## Purpose

`Text` is a polymorphic component that renders any element and inherits its
color from the parent by default. It maps typography scale tokens to a named
`variant` prop, and text-emphasis tones to a `color` prop, so you get consistent
type and hierarchy across the UI without repeating utility class strings.

## When to use

- Rendering headings, body copy, labels, captions, or code snippets that must
  follow the type scale.
- Applying alignment, truncation, or line-clamping without adding utility
  classes directly.

## When not to use

- Rich editorial content where each paragraph carries its own semantic element —
  write the HTML directly and style through the type scale CSS.
- Interactive text such as links or buttons — use the appropriate interactive
  component instead.

## Variants

The `variant` prop applies a typography scale token. Display steps are for large
hero text; body steps cover running copy; label and caption steps cover UI
chrome.

## Color

The `color` prop sets the text-emphasis tone. `inherit` (the default) takes the
parent color; `ink`, `body`, and `muted` step down the neutral emphasis ramp;
`danger` maps to its semantic role.

## Props

`Text` also accepts an `as` prop (default `"span"`) to render as a different
HTML element.

| Prop        | Type                                                                                                                                                                                                                                                                                                                  | Default     | Description                                                          |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------- | -------------------------------------------------------------------- |
| `variant`   | `"inherit" \| "display-xxl" \| "display-xl" \| "display-lg" \| "display-md" \| "display-sm" \| "body-lg" \| "body-lg-strong" \| "body-md" \| "body-md-strong" \| "body-sm" \| "body-sm-strong" \| "body-xs" \| "body-xs-strong" \| "label-lg" \| "label-md" \| "label-sm" \| "caption" \| "caption-strong" \| "code"` | `"inherit"` | Applies a typography scale token.                                    |
| `color`     | `"inherit" \| "ink" \| "body" \| "muted" \| "danger"`                                                                                                                                                                                                                                                                 | `"inherit"` | Sets the text-emphasis tone, inheriting the parent color by default. |
| `align`     | `"left" \| "center" \| "right" \| "justify"`                                                                                                                                                                                                                                                                          | —           | Sets the text alignment.                                             |
| `truncate`  | `boolean`                                                                                                                                                                                                                                                                                                             | `false`     | Cuts a single line of overflowing text with an ellipsis.             |
| `underline` | `boolean`                                                                                                                                                                                                                                                                                                             | `false`     | Underlines the text.                                                 |
| `lineClamp` | `1 \| 2 \| 3 \| 4 \| 5 \| 6`                                                                                                                                                                                                                                                                                          | —           | Limits the text to a fixed number of lines with an ellipsis.         |
| `className` | `string`                                                                                                                                                                                                                                                                                                              | —           | Additional CSS classes to apply.                                     |

## Styling

`Text` is styled through CSS classes, so the scale applies to any element that carries them.

| ------------------------------- | -------- | ------------------------------------------------------------------------ |

| `fri-text-inherit`              | Variant  | Inherits font and letter-spacing from the parent; no type scale applied. |
| `fri-text-display-xxl`          | Variant  | Display XXL type scale.                                                  |

| `fri-text-display-lg`           | Variant  | Display LG type scale.                                                   |
| `fri-text-display-md`           | Variant  | Display MD type scale.                                                   |
| `fri-text-display-sm`           | Variant  | Display SM type scale.                                                   |
| `fri-text-body-lg`              | Variant  | Body LG type scale.                                                      |
| `fri-text-body-lg-strong`       | Variant  | Body LG, strong weight.                                                  |
| `fri-text-body-md`              | Variant  | Body MD type scale.                                                      |
| `fri-text-body-md-strong`       | Variant  | Body MD, strong weight.                                                  |
| `fri-text-body-sm`              | Variant  | Body SM type scale.                                                      |
| `fri-text-body-sm-strong`       | Variant  | Body SM, strong weight.                                                  |
| `fri-text-body-xs`              | Variant  | Body XS type scale.                                                      |

| `fri-text-label-md`             | Variant  | Label MD type scale.                                                     |

| `fri-text-truncate`             | Modifier | Truncates one line with an ellipsis.                                     |
| `fri-text-underline`            | Modifier | Underlines the text.                                                     |

## Accessibility

- `Text` renders a `<span>` by default. Use the `as` prop to render a heading,
  paragraph, or other semantic element when the content requires it.
- The `truncate` and `lineClamp` props hide overflow visually but do not hide it
  from assistive technology, which reads the full text.
