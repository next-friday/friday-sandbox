import { Flex, Separator } from "@friday-sandbox/react";

## Purpose

The `Separator` draws a divider between sections of content. It wraps
[React Aria `Separator`](https://react-spectrum.adobe.com/react-aria/Separator.html),
so assistive technology announces it with the correct role.

## When to use

Use `Separator` to divide groups of related content, in a row or a column.

## When not to use

Do not use `Separator` for spacing alone. Set a layout gap for that.

## Orientation

Use `orientation="vertical"` to divide content laid out in a row. A vertical
separator takes its height from its container.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <Flex align="end" gap="md" className="px-4 py-6">
      <div style={{ display: "flex", alignItems: "center", height: 24, gap: 12 }}>
        <span>Edit</span>
        <Separator orientation="vertical" />
        <span>Duplicate</span>
        <Separator orientation="vertical" />
        <span>Delete</span>
      </div>
    </Flex>
  </Tab>

  <Tab value="Code">

  </Tab>
</Tabs>

## Props

`Separator` also accepts the props from React Aria `Separator`, excluding
`orientation`, which is the variant prop above, and a `className` string.

| Prop          | Type                         | Default        |
| ------------- | ---------------------------- | -------------- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |
| `className`   | `string`                     | —              |

## Accessibility

- Exposed with `role="separator"`, so assistive technology announces the
  division between regions.
- Set `orientation` to match the layout direction so the separator reports the
  correct orientation.
