## Purpose

The `Separator` draws a divider between groups of related content, in a row or a
column. It wraps React Aria `Separator`, so assistive technology announces it
with the correct role.

## When to use

- Divide groups of related content laid out in a row or a column.
- You want the division announced to assistive technology with the correct role.

## When not to use

- For spacing alone — set a layout gap instead.

## Orientation

Use `orientation="vertical"` to divide content laid out in a row. A vertical
separator takes its height from its container.

## Props

`Separator` also accepts the props from React Aria `Separator`, plus a
`className` string.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Sets the divider direction. |
| `className` | `string` | — | Additional CSS classes to apply. |

## Styling

`Separator` is styled through CSS classes, so the look applies to any element that carries them.

### Modifier classes

| Class                      | Type        | Description                                    |
| -------------------------- | ----------- | ---------------------------------------------- |
| `fri-separator`            | Base        | Base styles: horizontal default.               |
| `fri-separator-horizontal` | Orientation | A full-width hairline.                         |
| `fri-separator-vertical`   | Orientation | A full-height hairline sized by the container. |

## Accessibility

- Exposed with `role="separator"`, so assistive technology announces the
  division between regions.
- Set `orientation` to match the layout direction so the separator reports the
  correct orientation.
