## Usage

The `Separator` draws a divider between groups of related content, in a row or a
column. It wraps React Aria `Separator`, so assistive technology announces it
with the correct role. For spacing alone, set a layout gap instead.

## Examples

### Orientation

Use `orientation="vertical"` to divide content laid out in a row. A vertical
separator takes its height from its container.

## Props

`Separator` also accepts the props from React Aria `Separator`, plus a
`className` string.

| Prop          | Type                         | Default        | Description                          |
| ------------- | ---------------------------- | -------------- | ------------------------------------ |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Sets the divider direction.          |
| `className`   | `string`                     | —              | Additional CSS classes to apply.     |

## Accessibility

- Exposed with `role="separator"`, so assistive technology announces the
  division between regions.
- Set `orientation` to match the layout direction so the separator reports the
  correct orientation.

## Styling

`Separator` is styled through CSS classes ([source styles](https://github.com/next-friday/friday-sandbox/blob/main/packages/styles/src/components/bases/separator.css)), so the look applies to any element that carries them.

### Base class

- `fri-separator` — base separator styles with the horizontal default.

### Orientation modifier classes

- `fri-separator-horizontal` — a full-width hairline.
- `fri-separator-vertical` — a full-height hairline that takes its height from the container.
