## Purpose

The `Spinner` signals indeterminate progress, on its own or inside another
component such as `Button` while `isPending`. It renders a `<span>`.

## When to use

- Signal indeterminate progress, on its own or inside another component such as
  `Button` while `isPending`.

## When not to use

- For determinate progress with a known percentage — use a progress indicator
  instead.

## Props

`Spinner` also accepts the standard `<span>` props, plus a `className` string.

| Prop        | Default     | Type                                                                                                                      |
| ----------- | ----------- | ------------------------------------------------------------------------------------------------------------------------- |
| `color`     | `"primary"` | `"primary" \| "secondary" \| "accent" \| "info" \| "success" \| "warning" \| "danger"`<br />Sets the semantic color role. |
| `size`      | `"md"`      | `"xs" \| "sm" \| "md" \| "lg" \| "xl"`<br />Sets the spinner size.                                                        |
| `className` | —           | `string`<br />Additional CSS classes to apply.                                                                            |

## Styling

`Spinner` is styled through CSS classes, so the look applies to any element that carries them.

### Modifier classes

| Class                   | Type  | Description                          |
| ----------------------- | ----- | ------------------------------------ |
| `fri-spinner`           | Base  | Base styles: primary color, md size. |
| `fri-spinner-primary`   | Color | Primary color role.                  |
| `fri-spinner-secondary` | Color | Secondary color role.                |
| `fri-spinner-accent`    | Color | Accent color role.                   |
| `fri-spinner-info`      | Color | Info color role.                     |
| `fri-spinner-success`   | Color | Success color role.                  |
| `fri-spinner-warning`   | Color | Warning color role.                  |
| `fri-spinner-danger`    | Color | Danger color role.                   |
| `fri-spinner-xs`        | Size  | Extra small size.                    |
| `fri-spinner-sm`        | Size  | Small size.                          |
| `fri-spinner-md`        | Size  | Medium size (default).               |
| `fri-spinner-lg`        | Size  | Large size.                          |
| `fri-spinner-xl`        | Size  | Extra large size.                    |

## Accessibility

- The spinner is decorative; convey loading state in the surrounding context,
  such as a `Button` with `isPending` or text like "Loading…".
- If a standalone spinner is the only indication of a busy region, give that
  region `role="status"` and an accessible label so screen readers announce it.
