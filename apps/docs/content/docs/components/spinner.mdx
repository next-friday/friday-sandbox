import { Flex, Spinner } from "@friday-sandbox/react";

## Purpose

The `Spinner` signals indeterminate progress, on its own or inside another
component such as `Button` while `isPending`. It renders a `<span>`.

[Source](https://github.com/next-friday/friday-sandbox/blob/main/packages/react/src/components/bases/spinner/spinner.tsx) · [Style](https://github.com/next-friday/friday-sandbox/blob/main/packages/styles/components/spinner.css)

## When to use

- Signal indeterminate progress, on its own or inside another component such as
  `Button` while `isPending`.

## When not to use

- For determinate progress with a known percentage — use a progress indicator
  instead.

## Props

`Spinner` also accepts the standard `<span>` props, plus a `className` string.

| Prop        | Type                                                                                   | Default     | Description                      |
| ----------- | -------------------------------------------------------------------------------------- | ----------- | -------------------------------- |
| `color`     | `"primary" \| "secondary" \| "accent" \| "info" \| "success" \| "warning" \| "danger"` | `"primary"` | Sets the semantic color role.    |
| `size`      | `"xs" \| "sm" \| "md" \| "lg" \| "xl"`                                                 | `"md"`      | Sets the spinner size.           |
| `className` | `string`                                                                               | —           | Additional CSS classes to apply. |

## Styling

`Spinner` is styled through CSS classes ([source styles](https://github.com/next-friday/friday-sandbox/blob/main/packages/styles/components/spinner.css)), so the look applies to any element that carries them.

### Base class

- `fri-spinner` — base spinner styles with the primary color and medium-size defaults.

### Color modifier classes

- `fri-spinner-primary` · `fri-spinner-secondary` · `fri-spinner-accent` · `fri-spinner-info` · `fri-spinner-success` · `fri-spinner-warning` · `fri-spinner-danger`

### Size modifier classes

- `fri-spinner-xs` · `fri-spinner-sm` · `fri-spinner-md` · `fri-spinner-lg` · `fri-spinner-xl`

## Accessibility

- The spinner is decorative; convey loading state in the surrounding context,
  such as a `Button` with `isPending` or text like "Loading…".
- If a standalone spinner is the only indication of a busy region, give that
  region `role="status"` and an accessible label so screen readers announce it.
