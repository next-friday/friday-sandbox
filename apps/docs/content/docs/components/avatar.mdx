## Purpose

`Avatar` shows a user's image and falls back to initials or custom content when
the image is missing or fails to load. Compose `Avatar.Image` and
`Avatar.Fallback` inside `Avatar`, and overlap several with `Avatar.Group`.

## When to use

- A user's profile picture with an initials fallback for a missing image.
- A row of overlapping avatars for a shared resource, composed with
  `Avatar.Group`.

## When not to use

- A decorative image with no fallback need — use a plain `<img>`.
- A standalone status dot or count — compose it, rather than reaching for a
  whole avatar.

## Group

Overlap avatars with `Avatar.Group`. `size` cascades to the children, `ring`
(default on) separates them against the background, and `stacking` sets the
paint order. Compose a trailing `+N` avatar by hand for overflow.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <Flex align="center" gap="md" p="md">
      <Avatar.Group size="lg">
        <Avatar><Avatar.Fallback>A</Avatar.Fallback></Avatar>
        <Avatar><Avatar.Fallback>B</Avatar.Fallback></Avatar>
        <Avatar><Avatar.Fallback>C</Avatar.Fallback></Avatar>
        <Avatar><Avatar.Fallback>+5</Avatar.Fallback></Avatar>
      </Avatar.Group>
    </Flex>

## Fallback

`Avatar.Fallback` renders when no image is set or the image fails to load. Pass
initials or custom content as children, and use `delayMs` to avoid a flash on a
fast connection.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <Flex align="center" gap="md" p="md">
      <Avatar><Avatar.Fallback>AV</Avatar.Fallback></Avatar>
      <Avatar><Avatar.Fallback delayMs={300}>BR</Avatar.Fallback></Avatar>
    </Flex>
  </Tab>

  <Tab value="Code">
    ```tsx
    <Avatar>
      <Avatar.Image src="/broken.png" alt="Avatar image" />
      <Avatar.Fallback delayMs={300}>BR</Avatar.Fallback>
    </Avatar>
    ```

## Props

### Avatar Props

`Avatar` also accepts all props from the underlying Radix UI `Avatar.Root`,
such as `asChild`.

| Prop         | Type                                            | Default  | Description                                    |
| ------------ | ----------------------------------------------- | -------- | ---------------------------------------------- |
| `size`       | `"xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl"` | `"md"`   | Sets the avatar diameter.                      |
| `shape`      | `"square" \| "radius" \| "full"`                | `"full"` | Sets the corner treatment.                     |
| `isDisabled` | `boolean`                                       | `false`  | Dims an inactive avatar.                       |
| `className`  | `string`                                        | —        | Additional CSS classes to apply to the avatar. |

### Avatar.Image Props

`Avatar.Image` also accepts all props from the underlying Radix UI
`Avatar.Image`, such as `onLoadingStatusChange`.

| Prop        | Type     | Default | Description                                   |
| ----------- | -------- | ------- | --------------------------------------------- |
| `src`       | `string` | —       | The image source URL.                         |
| `alt`       | `string` | —       | Alternative text for the image. Required.     |
| `className` | `string` | —       | Additional CSS classes to apply to the image. |

### Avatar.Fallback Props

`Avatar.Fallback` also accepts all props from the underlying Radix UI
`Avatar.Fallback`.

| Prop        | Type     | Default | Description                                          |
| ----------- | -------- | ------- | ---------------------------------------------------- |
| `delayMs`   | `number` | —       | Delay before showing the fallback, to avoid a flash. |
| `className` | `string` | —       | Additional CSS classes to apply to the fallback.     |

### Avatar.Group Props

| Prop        | Type                                            | Default         | Description                                   |
| ----------- | ----------------------------------------------- | --------------- | --------------------------------------------- |
| `size`      | `"xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl"` | `"md"`          | Cascades to the children.                     |
| `ring`      | `boolean`                                       | `true`          | Draws a separating ring around each child.    |
| `stacking`  | `"first-on-top" \| "last-on-top"`               | `"last-on-top"` | Sets the overlap paint order.                 |
| `className` | `string`                                        | —               | Additional CSS classes to apply to the group. |

## Accessibility

- `Avatar.Image` requires `alt`; pass a meaningful description, or `alt=""`
  when the image is decorative.
- `Avatar.Fallback` renders readable initials or content when the image is
  missing; the swap is handled by Radix UI `delayMs` to avoid a flash.
- `isDisabled` is cosmetic — it dims the avatar but adds no ARIA state, since
  the avatar is not interactive.
