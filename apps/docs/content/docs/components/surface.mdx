## Purpose

The `Surface` paints the `--fri-surface` tokens onto a container — a card,
panel, or section — so background and border come from the theme instead of
hardcoded utilities. It is layout only: it sets no text color, and renders a
plain `<div>`, so it carries no interaction or role of its own.

## When to use

- Give a card, panel, or section a themed surface background.
- Inset the content with padding from the shared spacing scale.
- Round a container's corners with a step from the shared radius scale.

## When not to use

- No background is needed — lay content out with `Flex` or `Grid`.
- The container is interactive or pressable — reach for a button-like component.

## Variants

Use `variant` to set the surface fill: `primary` paints the base surface,
`secondary` the soft tier, and `outline` draws a border with no fill.

## Props

`Surface` also accepts the props of the native `div` it renders, plus a
`className` string.

| Prop        | Type                                                                       | Default     | Description                       |
| ----------- | -------------------------------------------------------------------------- | ----------- | --------------------------------- |
| `variant`   | `"primary" \| "secondary" \| "outline"`                                    | `"primary"` | Sets the surface fill.            |
| `radius`    | `"none" \| "xs" \| "sm" \| "md" \| "lg" \| "xl"`                           | `"md"`      | Rounds the corners with one step. |
| `p`         | `"2xs" \| "xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "3xl" \| "4xl"` | —           | Sets padding on all sides.        |
| `px`        | `"2xs" \| "xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "3xl" \| "4xl"` | —           | Sets horizontal (inline) padding. |
| `py`        | `"2xs" \| "xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "3xl" \| "4xl"` | —           | Sets vertical (block) padding.    |
| `pt`        | `"2xs" \| "xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "3xl" \| "4xl"` | —           | Sets top padding.                 |
| `pr`        | `"2xs" \| "xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "3xl" \| "4xl"` | —           | Sets right padding.               |
| `pb`        | `"2xs" \| "xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "3xl" \| "4xl"` | —           | Sets bottom padding.              |
| `pl`        | `"2xs" \| "xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "3xl" \| "4xl"` | —           | Sets left padding.                |
| `className` | `string`                                                                   | —           | Additional CSS classes to apply.  |

## Styling

`Surface` is styled through CSS classes, so the look applies to any element that carries them.

### Modifier classes

| Class                     | Type    | Description                                     |
| ------------------------- | ------- | ----------------------------------------------- |
| `fri-surface`             | Base    | Base styles: primary fill by default.           |
| `fri-surface-primary`     | Variant | Base surface fill.                              |
| `fri-surface-secondary`   | Variant | Soft surface tier.                              |
| `fri-surface-outline`     | Variant | Border with no fill.                            |
| `fri-surface-radius-none` | Radius  | Square corners.                                 |
| `fri-surface-radius-xs`   | Radius  | Extra-small corner radius.                      |
| `fri-surface-radius-sm`   | Radius  | Small corner radius.                            |
| `fri-surface-radius-md`   | Radius  | Medium corner radius.                           |
| `fri-surface-radius-lg`   | Radius  | Large corner radius.                            |
| `fri-surface-radius-xl`   | Radius  | Extra-large corner radius.                      |
| `fri-p-2xs` … `-p-4xl`    | Spacing | Shared padding on all sides, 2xs to 4xl.        |
| `fri-px-2xs` … `-px-4xl`  | Spacing | Shared horizontal (inline) padding, 2xs to 4xl. |
| `fri-py-2xs` … `-py-4xl`  | Spacing | Shared vertical (block) padding, 2xs to 4xl.    |
| `fri-pt-2xs` … `-pt-4xl`  | Spacing | Shared top padding, 2xs to 4xl.                 |
| `fri-pr-2xs` … `-pr-4xl`  | Spacing | Shared right padding, 2xs to 4xl.               |
| `fri-pb-2xs` … `-pb-4xl`  | Spacing | Shared bottom padding, 2xs to 4xl.              |
| `fri-pl-2xs` … `-pl-4xl`  | Spacing | Shared left padding, 2xs to 4xl.                |

## Accessibility

- `Surface` renders a plain container with no interaction or ARIA role; it does
  not trap focus or announce itself.
- `Surface` sets no text color — content inherits the page foreground; verify
  the content against the contrast floor on the fill you choose.
