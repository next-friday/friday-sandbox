## Install

Install the components and the styles package. React 19 and Tailwind CSS v4 are
peer dependencies.

```bash
pnpm add @friday-sandbox/react @friday-sandbox/styles
```

```bash
pnpm add react@^19 react-dom@^19 tailwindcss@^4
```

## Import the styles

The components are styled by class names defined in `@friday-sandbox/styles`.
Import the stylesheet once, at your application's CSS entrypoint, so Tailwind
compiles the design tokens and component layers:

```css
/* app.css */
@import "@friday-sandbox/styles";
```

`@friday-sandbox/styles` imports Tailwind CSS itself, so this is the only Tailwind
entrypoint you need.

## Use a component

```tsx
import { Button } from "@friday-sandbox/react";

export function Example() {
  return <Button>Save changes</Button>;
}
```
