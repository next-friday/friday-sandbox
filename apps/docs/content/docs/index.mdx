title: Introduction
description: An open-source UI component library for building accessible user interfaces.
---

`friday-sandbox` is an accessible React component library built on
`react-aria-components` and Tailwind CSS v4, paired with a framework-agnostic
design-token system. Components are styled through design tokens, so theming is
plain CSS variables with no build step.

- New here? Start with [Getting started](/docs/getting-started).
- Browse the [Components](/docs/components).
- Retheme with [Theming](/docs/theming).
