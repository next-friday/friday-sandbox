Friday Sandbox is a design system of accessible React components. Behaviour and
accessibility come from [React Aria Components](https://react-spectrum.adobe.com/react-aria/);
the visual layer is a token-driven Tailwind CSS v4 stylesheet shipped separately,
so the same theme works in any framework that can load plain CSS.

## Packages

- **`@friday-sandbox/react`**: the React components.
- **`@friday-sandbox/styles`**: design tokens and the Tailwind CSS v4 layers the
  components are styled with. It is a peer dependency of the React package.

## Highlights

- Accessible by default: keyboard interaction and ARIA handled for you.
- Themed through plain CSS custom properties and a `data-theme` contract, with
  light and dark out of the box.
- Typed props with small, focused component APIs.
