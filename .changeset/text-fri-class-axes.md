Wrapped `Text`'s `align`, `lineClamp`, `truncate`, and `underline` axes in `fri-text-*` classes (`fri-text-align-<value>`, `fri-text-clamp-<n>`, `fri-text-truncate`, `fri-text-underline`), so every axis participates in the framework-agnostic class contract; the raw Tailwind utilities are no longer emitted.
