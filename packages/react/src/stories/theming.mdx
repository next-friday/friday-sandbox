# Theming

## Toggle dark mode

Add the class or attribute to any element (typically `<html>`):

```html
<html class="dark">
  …
</html>
<!-- or -->
<html data-theme="dark">
  …
</html>
```

Default is light. The toggle is scopable: apply `.dark` to a subtree to flip just that subtree.

The Storybook toolbar at the top of any story exposes the same switch under the **Theme** control.

## Customize brand colors

Override any intent in your app's CSS. Components recompute readable text contrast and hover state automatically. The values below are the real light-mode defaults, so this block reproduces the shipped theme as a starting point:

```css
:root {
  --primary: oklch(57.49% 0.2084 257.52);
  --secondary: oklch(43.92% 0.0192 239.51);
  --accent: oklch(0 0 0);
  --neutral: oklch(65.47% 0.0208 248.15);
  --info: oklch(55.53% 0.175 255.24);
  --success: oklch(53.24% 0.1205 159.6);
  --warning: oklch(73.3% 0.1596 63.45);
  --danger: oklch(56.13% 0.1887 21.51);
}
```

Tip: stick to clearly light or clearly dark colors. A mid-tone value around OKLCH `L 60%` can pick a foreground that feels low-contrast.

## Customize surfaces

These are the real default surface values. The dark block swaps them under `.dark`:

```css
:root {
  --background: oklch(100% 0 0);
  --foreground: oklch(0 0 0);
  --muted: oklch(89.2% 0.0051 228.83);
}

.dark {
  --background: oklch(18% 0 0);
  --foreground: oklch(95% 0 0);
  --muted: oklch(30% 0.006 228.83);
}
```

## Customize sizing

Each scope's base unit and radius can be retuned independently:

```css
:root {
  --size-action: 0.3rem; /* scale every action component up */
  --radius-action: 9999px; /* pill-shaped buttons */
  --radius-field: 0.25rem; /* squarer inputs */
  --radius-box: 1rem; /* rounder cards */
}
```

Override `--size-{scope}` and every size from `xs` through `xl` rescales together, so you do not have to touch each variant.

`--radius-{scope}` is the reference at `md`. Smaller sizes get proportionally smaller corners, larger sizes get larger ones, so a pill never collapses to a square at `xs` or stretches at `xl`.
