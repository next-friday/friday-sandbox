<Subtitle>
  Accessible React UI components themed with CSS variables.
</Subtitle>

Components ship with keyboard navigation, focus rings, hover, and press states built in. Colors, sizes, and corner radius are CSS variables — override one and every component updates.

## Install

```sh
pnpm add @friday-sandbox/react @friday-sandbox/styles
```

Peer dependencies: `react@^19`, `react-dom@^19`, `tailwindcss@^4`.

## Usage

Import the stylesheet once at the application root:

```ts
import "@friday-sandbox/styles";
```

Then import components:

```tsx
import { Button } from "@friday-sandbox/react";

export function Example() {
  return <Button variant="primary">Click me</Button>;
}
```

## Theming

Switch themes via the `data-theme` attribute on `<html>`:

```html
<html data-theme="light">
  <!-- or "dark" -->
</html>
```

The Storybook toolbar exposes the same switch under the **Theme** control. See the [Theming](?path=/docs/theming--docs) page for color, surface, and sizing overrides.
